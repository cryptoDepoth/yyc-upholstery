## Design System Requirements

### Target Audience
- Calgary homeowners (families, young professionals, seniors)
- Local businesses (offices, hotels, rental properties)
- Pet owners, allergy sufferers

### Style & Tone
- **Trustworthy & Professional** — but friendly, approachable
- **Local Pride** — Calgary-focused messaging
- **Clear & Reassuring** — address concerns about mess, drying time, safety
- **Minimalist SEO Design** — clean, fast-loading, mobile-first

### Color Palette
- **Primary:** Royal Blue `#2563eb` (trust, professionalism)
- **Secondary:** Orange `#f97316` (energy, action, warmth)
- **Background:** White `#ffffff`
- **Text:** Dark Gray `#333333`
- **Accent:** Soft Yellow `#fffbec` (callout boxes)

### Typography
- **Headings:** System UI sans-serif (Apple San Francisco / Segoe UI)
- **Body:** System UI sans-serif
- **Size:** 16px base, responsive scaling
- **Line Height:** 1.6 for readability

### UI Components

#### Hero Section
- Bold heading ("Calgary's Trusted Upholstery")
- 2 CTAs: DM (Instagram), Call Now
- Background: gradient blue
- Mobile-first responsive layout

#### Service Cards
- Clear icons + title + description
- Localized keywords (e.g., "sofa cleaning Calgary SW")
- Price range shown

#### Community/Neighborhood Pages
- Local neighborhood focus (Airdrie, Cochrane, etc.)
- Map coordinates, local testimonials
- Same-day service guarantee

#### Pricing
- Tiered bundles (Single Item, Living Room, Mattress, Car)
- Clear "No hidden fees" messaging
- Real Calgary prices
