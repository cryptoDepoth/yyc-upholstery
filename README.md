## 🛠️ Setup
1. Clone repo: `git clone https://github.com/YOUR_USERNAME/yyc-upholstery.git`
2. Install Wrangler: `npm install -g wrangler`
3. Login: `wrangler login`
4. Deploy: `wrangler pages deploy public --project-name yycupholstery-site`
